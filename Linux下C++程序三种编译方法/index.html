Linux下C++程序编译方法（一）

发表于2019-03-15|C++

问题

西安出租车的收费标准：起步价（2公里以内，含2公里）为7元钱，超过2公里且在15公里以内（含15公里）时每公里收费1.5元，超过15公里时每公里收费1.2元，不足1公里按照1公里收费。

输入输出

  • 输入
    输入为实型（单位为公里）
  • 输出
    输出为实型，保留六位小数（单位为元）

输入示例
9.2
输出示例
19.000000

编译方法

g++

在源码所在文件夹右键>>在终端打开，输入如下：
g++ jifei.cpp -o jifei

就生成一个可执行的文件 (application/x-executable)

再输入：
./jifei

就会在终端启动可执行文件。
c文件同样在gcc或g++中进行编译

make

用make命令执行会调用同目录下的makefile文件作为make的规则，是以用make进行编译时需要编写一makefile文件。
在目录新建无后缀名为makefile或Makefile文件，进入编写。

保存后即可在终端输入：
make

编译生成一个jifei的可执行文件，输入：
./jifei

即可运行程序。

cmake

  1. 写CMakeLists.txt文件
  2. 终端输入：
    cmake_gui
    打开cmake，选中CMakeLists.txt所在文件夹作为Source，在其目录下新建build文件夹作为Build的目录，>>Configure>>Generate，即可在build下自动生成一Makefile文件。终端在build目录下输入：
    make
    则生成一可执行文件jifei，输入：
    ./jifei
    即可运行程序。
