  2. 配置好github的ssh等内容。

主题

  1. 安装可使用的主题，我使用butterfly主题对博客进行配置。

    在Hexo根目录执行命令克隆主题内容。

  2. 安装pug和stylus渲染器插件。

  3. 文件夹下的_config.yml文件，更改以下内容，应用主题。

  4. 未来减少升级带来的不便，可在 hexo 的根目錄創建一個文件 _config.butterfly.yml，並把主題目錄的 _config.yml 內容複製到 _config.butterfly.yml 去。( 注意: 複製的是主題的 _config.yml ，而不是 hexo 的 _config.yml)

    hexo会自动合并两个配置文件。
