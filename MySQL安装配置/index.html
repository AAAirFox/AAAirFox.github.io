MySQL+Navicat安装配置

发表于2023-09-24|后端

参考链接：MySql 安装与使用（非常详细）

Mysql一个关系型数据库管理系统，Navicat一个数据库管理工具，提供可视化界面用于操作MySQL数据库管理系统。

下载

MySQL官网：https://dev.mysql.com/downloads/installer/

Navicat官网：https://www.navicat.com.cn/

安装

MySQL

server

  1. 官网选择平台下载。

  2. 打开安装。

  3. 安装server

  4. 一路next，设定root密码，同时添加一个用户。

  5. 继续next，最后安装

  6. 打开刚刚安装上的line client，输入密码，连接成功服务器。

add

  1. 选择Add ...增加要安装的内容。

    Filter中可选择的Maturity中，Current Bundle是当前版本的打包，Current GA是正式发布的稳定版本。

    • MySQL Servers 是安装 MySQL 数据库，安装数据库只需要在这里寻找安装。

    • Applications 是为集成到其他应用提供的插件，可以不装，需要时再装。

      MySQL Workbench：ER（实体关系）和数据库建模工具。

      MySQL Shell：类似SQL功能、JavaScript和Python脚本功能 + API，可以方便地进行搭建、管理和配置。

      MySQL Router：实现负载均衡和高可用性。

    • MySQL Connectors 是MySQL 连接器，为不同的编程环境提供的连接支持，可以之后再安装。

    • Documentation 会有文档以及一些案例，类似于帮助手册。安装数据库的时候勾选即可。

  2. 选择安装Applications、Documentation和C++ Connector，同时选择自定义安装选项、更改安装路径。

  3. next下载。

  4. 自定义选择安装。

  5. 继续next开始安装。

  6. 连接服务器。
