Shell

发表于2019-07-05

概念

  1. 什么是shell？

    是一个程序，是一个命令行解释器，是与操作系统内核交互的接口。shell脚本是一种编程语言，用于编写脚本程序，通常用于自动化重复性任务，比如文件操作、系统管理和数据处理。

  2. 和终端有什么区别？

    终端是访问操作系统的用户界面，shell是一种为操作系统提供命令行界面的计算机程序。

  3. 什么是shell脚本？

    shell脚本是一种为shell编写的脚本程序，实际上是一系列命令的集合，按照特定的顺序和逻辑组织在一起，以实现某种特定的功能或任务。

  4. shell变量和环境变量有什么区别？

    • 作用域

      shell变量是本地变量，仅限于当前实例或子进程。

      环境变量可以被子进程继承，在多个进程和会话之间共享。

    • 生命周期

      shell变量仅在当前实例或子进程存在时有效，终止时销毁。

      环境变量在实例及子进程中持续存在，知道被显示修改。

    • 设置方式

      shell变量为variable_name=value；

      环境变量为export variable_name=value.

  5. 子shell和子进程有什么区别？

    子shell是父shell的一个复制，可以访问父shell的变量。
