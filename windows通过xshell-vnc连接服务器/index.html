windows通过xshell+vnc连接服务器

发表于2023-01-25|服务器操作

下载

xshell：XSHELL7官方网址，下载即可。

xftp：XFTP7官方网址，下载即可。

RealVNC：RealVNC官方地址，下载对应版本。

安装

XShell | XFtp

默认安装即可。

RealVNC

  1. 点击exe文件，默认下一步安装即可。

    右上角点击选择option

  2. 最重要的两步:

    • 选择按照VNC password登录，然后点击apply/ok

    • 设置密码：最少6位，然后点ok

  3. 进入VNC Server安装目录

    打开vnclicensewiz ，选择第三项Enter a license key ... 找个 KEY 填上

    点击apply–>done完成

  4. 完成后，打开 CMD , 输入

  5. 使用客户端连接测试

    使用ip和vnc密码登录（无需端口）

连接

  1. 选择新建

  2. 填写连接主机号等。

  3. 填写用户名密码。

  4. 要使用ftp进行文件传输，可通过xshell直接调用xftp应用。

    点击xftp图标：

  5. 在xshell中打开vnc接口。

    显示当前存在的接口。

  6. 新开一个vnc接口。

    初次需要设置vnc密码，长度6到8位；然后选择n.

  7. 记住自己的端口号（我的在这里面是1），在vnc view中输入[host ip]:1

  8. 即可进入界面操作。

错误处理

错误：连接VNC后屏幕是灰色的。

  1. 删除之前的端口（若没有开则直接执行第二步）：

    vncserver -kill :端口号

  2. 备份~文件夹下重要的隐藏文件, 如.bashrc文件：

    cp ~/.bashrc ~/bashrc.bak

  3. 删除~文件夹下所有隐藏文件

    rm -rf ~/.*

  4. 复制文件至usrname下面（你自己的用户名）

    cp -r /etc/skel/. /home/usrname

  5. 输入exit登出账户, 再重新登陆ssh

  6. 修改xtartup配置，终端输入

    vim ~/.vnc/xstartup

  7. 按i进行编辑，将原内容删除，粘贴如下内容，按Esc，输入“:wq”保存并退出：

  8. vncserver -geometry 1920x1080激活新vnc窗口
